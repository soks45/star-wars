# StarWars

## Требования
Реализовать CSR (Client Side Rendering) SPA (Single Page Application) приложение на Angular.
В приложении должно быть две страницы:
1. Список персонажей с пагинацией и поиском;
2. Карточка персонажа с возможностью редактирования без отправки на сервер.

Обязательно:
* В качестве источника данных SWAPI (https://swapi.dev или https://swapi.py4e.com/);
* Typescript;
* UI-фреймворк;
* Аккуратная верстка;
* Деплой.
